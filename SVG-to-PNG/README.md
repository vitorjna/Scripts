# SVG to PNG Converter

A simple script to convert SVG files to PNG format.

## Features
- Drag and drop support for multiple SVG files.
- Default resolution of 1024x1024.
- User-definable resolution during conversion.

## Installation
Install the dependencies using the provided requirements file:
```bash
pip install -r requirements.txt
```
Alternatively, you can install them manually:
```bash
pip install svglib reportlab
```
